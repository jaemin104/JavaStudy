# 자바 공부하기

## 인텔리제이 단축키 정리

* 한줄 올리기/내리기 : 옵션 + 쉬프트 + 화살표 위/아래
* 한줄 복사 : 커맨드 + D
* 찾기/바꾸기 : 커맨드 + F
* 변수명 한 번에 바꾸기 : fn + shift + f6
* 줄맞춤 : 옵션 + 커맨드 + L
